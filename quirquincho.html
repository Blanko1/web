¿Qué es Quirquincho?

Quirquincho es un Bot que permite conectar la Red Chaucha con el grupo de Telegram del Proyecto Chaucha, y fue nombrado en honor al armadillo peludo que habita el altiplano de la Cordillera de los Andes.

Este proyecto nació de manera accidental, en base a una pelea interna entre los desarrolladores del Proyecto Chaucha para decidir la manera correcta de diseñar un Bot para la comunidad.

Listado de comandos

La comunicación entre el usuario y Quirquincho se realiza a partir de comandos específicos para cada tarea a desempeñar, y pueden ser ejecutados de manera privada (hablando directamente con @QuirquinchoBot) o pública (dentro del grupo de Proyecto Chaucha).

Cada comando dentro del sistema tiene asignado una sintaxis de respuesta en formato de texto, que es conformada en base a la información presente en la Red Chaucha.

A continuación se presenta la lista completa de comandos que interpreta Quirquincho, junto con una pequeña explicación de su funcionamiento, la sintaxis para ejecutar correctamente el comando y un ejemplo simple de pregunta y respuesta.


Online Wallet

/address
Creación de dirección dentro de la wallet online de Quirquincho, que se genera de manera privada y que solo es posible interactuar con el usuario de Telegram que utilizó el comando.

Respuesta: {address}


/balance
Saldo disponible de Chauchas que pertenece al usuario.

Respuesta: {balance} CHA


/send X Y
Generación de transacción dentro de la Red Chaucha, que envía X Chauchas del saldo disponible del usuario hacia la dirección Y.

Respuesta: {Transaction ID}



Información

/red
Funcionamiento actual de la Red Chaucha.

Respuesta: {Height} {Difficulty} {Hashrate} {Time between blocks}


/precio
Variación del precio de la Chaucha en SouthXchange.

Respuesta: {Ask} {Bid} {Variation24Hr}



Juego

/dice X
Quirquincho calcula un número N aleatorio entre 0 y 1000.

Si N > 500 el usuario gana X CHA
Si N < 500 el usuario pierde X CHA
si N = 500 el usuario gana X*2 CHA
si N = X no sucede nada

Respuesta: {N} {Result}


/info
Saldo perteneciente a Quirquincho, que es utilizado para pagar el premio del comando anterior.

Respuesta: {Balance}



Ayuda

/help
Pequeño tutorial de utilización de la Wallet Online.

Respuesta: {Tutorial}
