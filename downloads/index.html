Como minar Chauchas

    La Red Chaucha utiliza el algoritmo de distribución Proof of Work (PoW), que entrega una cantidad definida de Chauchas al usuario que logre resolver un puzzle criptografico a traves de un proceso de fuerza bruta. Este proceso es llamado comúnmente como minar una criptomoneda.

    El mayor problema de este algoritmo es la competencia, a partir de que usuarios con bajo poder de cómputo nunca podrán resolver un bloque por si mismos, y por ende nunca podrán obtener el botín que genera la red.

    Para resolver este problema, se utiliza una herramienta llamada Mining Pool, que concentra todo el poder de minado de múltiples usuarios en un solo cliente que mina un bloque, a modo de asegurar que la organización obtenga la ganancia del bloque, para luego ser distribuida entre los usuarios en base a su participación en el minado del bloque.
Mining Pool oficial del Proyecto Chaucha
Algoritmo: Scrypt
IP: pool.chaucha.cl
Puerto: 3008 (CPU) / 3032 (GPU/ASIC)

    Si deseas minar Chauchas debes conectarte a la Mining Pool del proyecto con el programa de minado que soporte tu máquina. En la actualidad existe una gran cantidad de alternativas de software capaces de permitir la conexión a las Mining Pool para ofrecer tu poder de cómputo de GPU y CPU.

    A modo de agradecimiento a los primeros "mineros" de la Red Chaucha, se modificó el algoritmo de generación de criptomonedas para entregar una mayor ganancia en los primeros 50.000 bloques creados. La fórmula para calcular la ganancia por bloque es el número entero de (49990/n + 10), y por esto, a partir del bloque #50001 se fijará la recompensa en 10 CHA por bloque minado.
