Descargas

    Chauchera es el cliente utilizado para la conexión a la Red Chaucha, que permite almacenar tus Chauchas de manera segura. Este cliente fue compilado exclusivamente para sistemas de 64 bits.

Código Fuente

    El codigo fuente de Chauchera fue alojado en los servidores de Gitlab para proteger el desarrollo del Proyecto Chaucha de un Bot ruso que intentó robarse los primeros bloques generados por la red. En estos momentos la red ya entró a una fase estable de minado, pero de todas maneras dejaré alojado el código fuente en gitlab.

Repositorio de Chauchera


    Si deseas utilizar la versión de consola de Chauchera, o deseas modificar el codigo fuente para mejorar tu experiencia en la Red Chaucha, te recomiendo compilar desde la distribución Ubuntu 14.04, porque da menos problemas de compatibilidad.
